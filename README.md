## About

**Cosmic Themes** is a collection of beautifully crafted, space-inspired themes for Discord. Each theme is designed to bring the wonder and mystery of the cosmos right to your chat interface.

Whether you're a fan of deep nebulae, distant galaxies, or the serene glow of starlight, there's a theme here that will make your Discord feel truly out of this world.

## Installation

### BetterDiscord
1. Download the `.theme.css` file of your choice from the `/themes` folder
2. Open Discord and go to **User Settings** -> **BetterDiscord** -> **Themes**
3. Click **Open Theme Folder**
4. Drag and drop the theme file into the folder
5. Toggle the theme **ON**
